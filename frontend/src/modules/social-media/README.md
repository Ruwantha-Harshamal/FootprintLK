# Module 2: Social Media Exposure

## Purpose
Discovers what personal information about the user is publicly visible across social media platforms.

## Features (Planned)
- Cross-platform profile search
- Public data aggregation
- Network graph visualization
- Privacy risk assessment
- Platform-specific recommendations

## Folder Structure
